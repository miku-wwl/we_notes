| 场景描述                | 原字符/动作       | Terraform 转义写法 |
|-------------------------|-------------------|--------------------|
| Literal Quote（双引号） | `"`（双引号）| `\"`               |
| Literal Backslash（Windows 路径反斜杠） | `\`（反斜杠）| `\\`               |
| New Line（换行）| 换行（动作）| `\n`               |
| Tab Character（制表符） | Tab 缩进（动作）| `\t`               |
| new Line（制表符+换行） | 换行（动作） | `\n`（换行） |
