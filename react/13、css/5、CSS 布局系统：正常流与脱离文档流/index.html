3. 定位（Positioning）

static（默认定位）- 正常流中的元素
relative（相对定位）- 相对于自身正常位置偏移
absolute（绝对定位）- 相对于最近的定位祖先元素定位
fixed（固定定位）- 相对于视口定位，滚动时位置不变
sticky（粘性定位）- 滚动到一定位置时固定
