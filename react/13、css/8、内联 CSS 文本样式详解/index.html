字体、大小、行高与对齐的直观展示

1. 字体家族 (font-family)

无衬线字体 (Sans-serif)：Arial, Helvetica

衬线字体 (Serif)：Georgia, Times New Roman

等宽字体 (Monospace)：Courier New, Monaco

手写体 (Cursive)：Brush Script MT

2. 字体大小 (font-size)

超小文本 (12px)

小文本 (14px)

基础文本 (16px - 默认)

大文本 (18px)

超大文本 (24px)

特大文本 (32px)

相对单位文本 (1.2rem)

3. 字体样式与粗细

正常粗细 (font-weight: normal)

粗体文本 (font-weight: bold)

轻量文本 (font-weight: 300)

斜体文本 (font-style: italic)

倾斜文本 (font-style: oblique)

uppercase (全部大写)

LOWERCASE (全部小写)

capitalize each word (首字母大写)

4. 行高与字间距

紧凑行高 (line-height: 1.2) - 这段文本用于演示较紧凑的行高效果，适合标题等短文本展示。

正常行高 (line-height: 1.6) - 这段文本用于演示正常的行高效果，适合大多数正文内容，提供良好的可读性。

宽松行高 (line-height: 2.0) - 这段文本用于演示宽松的行高效果，适合长段落阅读，减少视觉疲劳。

字间距紧凑 (letter-spacing: -0.5px)

正常字间距 (letter-spacing: 0)

字间距宽松 (letter-spacing: 2px)

词间距宽松 (word-spacing: 8px) - this demonstrates wide word spacing

5. 文本对齐 (text-align)

左对齐文本 (text-align: left) - 这是默认的文本对齐方式，适合大多数语言的阅读习惯。

居中对齐文本 (text-align: center) - 常用于标题和强调内容，使文本在容器中水平居中。

右对齐文本 (text-align: right) - 常用于日期、金额等需要靠右显示的内容。

两端对齐文本 (text-align: justify) - 使文本左右两端都对齐，常用于报纸和书籍排版，使页面看起来更整齐。这种对齐方式会自动调整单词间距，使每行的长度相等（最后一行除外）。

6. 文本装饰与效果

下划线文本 (text-decoration: underline)

上划线文本 (text-decoration: overline)

删除线文本 (text-decoration: line-through)

带阴影的文本 (text-shadow)

首行缩进文本 (text-indent: 2em) - 这段文本展示了首行缩进效果，常用于段落开头，符合中文排版习惯。
