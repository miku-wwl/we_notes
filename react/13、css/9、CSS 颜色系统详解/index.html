CSS颜色系统详解

色值表示方法与Material UI主题色

1. 色值表示方法

关键词表示

style="background-color: red;"

使用预定义的颜色名称，如red、blue、green等

十六进制表示

style="background-color: #ff3e3e;"

#RRGGBB格式，范围00-FF，可简写#RGB

RGB表示

style="background-color: rgb(52, 152, 219);"

red, green, blue三通道，范围0-255

RGBA表示

style="background-color: rgba(52, 152, 219, 0.5);"

RGB+Alpha透明度，0-1之间

HSL表示

style="background-color: hsl(120, 60%, 50%);"

色相(0-360)、饱和度(0-100%)、亮度(0-100%)

HSLA表示

style="background-color: hsla(120, 60%, 50%, 0.5);"

HSL+Alpha透明度，0-1之间
