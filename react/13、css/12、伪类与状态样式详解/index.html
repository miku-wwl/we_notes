伪类与状态样式详解

:hover、:active等伪类及Material UI组件状态

1. 常用状态伪类

伪类用于定义元素在特定状态下的样式，常用状态伪类包括：

  • :hover - 鼠标悬停在元素上时
  • :active - 元素被激活（如点击时）
  • :focus - 元素获得焦点时（如输入框）
  • :disabled - 元素被禁用时
  • :visited - 链接被访问后
